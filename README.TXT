PROJECT TITLE: 
Gestión de Tareas - Implementación de Patrones de Diseño en Java
PURPOSE OF PROJECT:
Demostrar la aplicación práctica de tres tipos de patrones de diseño (Creacional, 
Estructural y de Comportamiento) usando Java, mediante un sistema de gestión de tareas 
por consola.
VERSION or DATE:
HOW TO START THIS PROJECT:
AUTHORS:Romane Jachacollo Montaño
USER INSTRUCTIONS:

La aplicación crea tareas y las gestiona desde la terminal.

Se agregan tareas normales y urgentes utilizando una fábrica y decoradores.

Se usan comandos para ejecutar acciones como agregar o completar tareas.

El sistema imprime en pantalla la lista de tareas con su estado (completadas o no).
